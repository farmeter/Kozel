# Kozel
Simple Commerce Web Project by farmeter

## Environment
Jdk 11, React

## Build
```
gradle build
```

## Run
```
$ cd kozel-web
$ yarn start
```

```
$ cd kozel-api
$ gradle bootRun
```

using Intellij 
 - check "Project Structure > Project SDK, Project Language Level"
 - check "Preference > Build.. > Build Tools > Gradle > Gradle JVM"
